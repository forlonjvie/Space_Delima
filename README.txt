Setup and Running the Application
Download or Clone the Repository:
Download the zip file or clone the repository to your local machine using the following command:
git clone <repository-url>

Obtain an API Key:
Visit NASA's API portal to request your own API key. This key will allow you to access NASA's data.

Run the Application:
If you are running the application locally, navigate to the directory where you downloaded or cloned the repository and open index.html in your browser. The path will typically be localhost/Space_Delima/index.html.

Steps to Obtain and Use the API Key
Visit the NASA API Portal:
Go to https://api.nasa.gov/.

Sign Up for an API Key:
Fill in the necessary fields, including your first name, last name, email, and the purpose for which you need the API key. Ensure you use a valid personal email address.

Check Your Email:
After completing the sign-up process, check your email inbox for a message from NASA. The email will contain your generated API key.
